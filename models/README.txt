This folder stores generated and downloaded model files:

1. encodings.pkl       — auto-generated by encode_faces.py
2. deploy.prototxt     — (optional) DNN face detector config
3. res10_300x300_ssd_iter_140000.caffemodel — (optional) DNN weights

encodings.pkl is excluded from git (see .gitignore).
Download DNN files from the OpenCV samples repo if needed.
